Recent Projects


Synthetic dataset generation for machine learning

The objective of this project is to generate fake datasets for training machine learning models. The object is scanned using a depth camera and a 3D model is generated. Then millions of fake images are generated using the 3D model and the images are added with a variety of backgrounds, to generate a variety of data to train robust detection and classification models.

Skills used:

PCL, OpenCV, tensorflow, Open3D, Gazebo, C++, Python
Autonomous Drone for Agriculture

This is a work in progress and is my final MS Robotics project which is to be completed by end of Fall 2018. The objective of this project is to develop an autonomous drone that can traverse through a field give GPS co-ordinates as bounding box. This involves visual odometry, sensor fusion, controls etc and RTK GPS is used to obtain centimeter level accuracy. The video demonstrates the final objective of the project.
Hopp3r: Single legged planar hopping robot

The objective of this project was to design a hopping robot for a NASA project that investigates legged locomotion on soft ground.The robot is planar, mounted on a boom, with 3 parallel DOF. This was developed for as a part of a 2 quarter long coursework and is the combined effort of 7 team members.

Excavator tooth detection

This project is an attempt to train a custom model to detect and locate each tooth on a excavator bucket. Tensorflow machine learning library from Google was used to train the custom model along with OpenCV. The model was trained with just 100 images for this project, and so the accuracy of detection can be increased by increasing the number of training images and fine tuning the parameters between accuracy and speed.

Baxter: Constrained motion

The objective of this project was to enable baxter to find a cup on the table using camera vision and IR Sensor, then handle the cup to the right hand of the person in front of him, which was sensed by applying skeleton tracking on pointcloud data from a depth camera .
Quadcopter from scratch

The objective of this project was to develop a quadcopter from scratch. PID controllers was used to control the roll, pitch and yaw of the quadcopter, using the input from the onboard IMU. Finally the quadcopter was able to fly autonomously and hold its position in air without drifting using HTC Vive light house as external sensor.
